# OfficeHub

OfficeHub is a web-based Office Equipment Shopping and Inventory Management System developed using PHP and MySQL. The system enables customers to browse and purchase office equipment while allowing administrators to manage products, inventory, users, and system activities through a secure dashboard.

## ✨ Features

### Guest
- Home Page
- About Page
- User Registration
- Email Verification via Gmail SMTP
- Secure Login

### Buyer
- Browse Products
- Shopping Cart
- Checkout
- Cash on Delivery (COD)

### Administrator
- Dashboard
- Product Management
- User Management
- Inventory Management
- Audit Logs

## 🔒 Security Features

- Password Hashing
- Session-Based Authentication
- Email Verification using PHPMailer
- Unique Verification Tokens
- Role-Based Access Control

## 🛠️ Technologies Used

- MySQL
- HTML5
